MindCorp is a corporate management portal built on ASP.NET Core Razor Pages. It provides a complete security administration layer — letting administrators manage user accounts, role profiles, system modules, and granular per-action permissions — all through a clean, responsive web interface.

## What MindCorp provides

<CardGroup cols={3}>
  <Card title="Secure Access" icon="key">
    JWT authentication with Cloudflare Turnstile CAPTCHA and cookie-based sessions
  </Card>
  <Card title="Role-Based Control" icon="users-gear">
    Fine-grained permissions per profile: view, create, edit, delete, and detail access per module
  </Card>
  <Card title="Full CRUD Admin" icon="table-list">
    Complete management screens for Modules, Profiles, Users, and Profile Permissions
  </Card>
  <Card title="Dynamic Navigation" icon="bars">
    The sidebar and top menu adapt automatically based on the logged-in user's permissions
  </Card>
  <Card title="API-Backed" icon="plug">
    All data flows through the companion WebCoorporativaAPI, keeping the frontend stateless
  </Card>
  <Card title="Responsive Design" icon="mobile-screen">
    Neumorphic Bootstrap 5 UI that works on desktop and mobile
  </Card>
</CardGroup>
